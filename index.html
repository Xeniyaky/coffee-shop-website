Welcome to Brew Haven

This project is a static website prototype for a coffee shop management system. It is designed to help manage customer orders, menu items, inventory, and sales data.

The website includes CRUD pages, interactive charts, an About Us page, and a Botpress chatbot that helps users with coffee shop questions and customer support.
